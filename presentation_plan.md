# План презентации: Инфраструктура Momo Store

## 1. Введение
- Обзор проекта: Магазин пельменей "Momo Store".
- Стек технологий: Go (backend), Vue.js (frontend), Nginx.

## 2. Инфраструктура как код (IaC)
- Использование Terraform для управления ресурсами в Yandex Cloud.
- Обзор создаваемых ресурсов: VPC, Managed Kubernetes, S3 (для статики и стейта), Container Registry.
- Демонстрация хранения стейта в S3.

## 3. CI/CD Пайплайн
- Описание этапов пайплайна в GitLab CI.
- Тестирование: Unit-тесты для Go.
- Сборка Docker-образов (multi-stage) и пуш в реестр.
- Хранение артефактов (Helm-чарты) в Nexus.

## 4. Оркестрация (Kubernetes)
- Развертывание приложения с помощью Helm.
- Настройка Ingress и автоматическое получение TLS сертификатов (cert-manager).
- Масштабирование и управление конфигурацией через values.yaml.

## 5. Мониторинг и логирование
- Сбор метрик с бэкенда (Prometheus клиент).
- Использование Grafana для визуализации состояния приложения и кластера.

## 6. Заключение
- Итоги реализации: автоматизированный процесс от коммита до деплоя.
- Безопасность: использование секретов GitLab CI, разделение прав доступа.
